This is a mashup site with some of my bike rides: Strava GPS stats & pictures:


  • My Strava stats below are compiled by using the Strava API in my local dev environment, which is running Node.js and XAMPP. The 'video' at the bottom of this page is a data viz created from the GPS file data from one of my recent rides.
    • This site was quickly developed using the Joomla Content Management System (CMS) with the JaxStorm-Black template, which uses SlidesJS Jquery plugin for the image gallery effects. I built a custom joomla module in object oriented PHP to get the date from the page name in the URL and simply match it with all photos in a directory which are named with the same date. The resulting images array then gets passed to the SlidesJS output. Voilà, dynamique!

Why Hello, Sports Fans...

  • Collecting Strava KOMs in 2021 required entering a mental space which became tiresome to me, like work. There were no immediate wildfires here, but it was hotter and drier than normal, with little wind for motivation. So I just focused on putting in miles, as the majority of my rides were 50 milers in a span of 3 hours. So, I ended up with just over 4,000 miles for this year.
